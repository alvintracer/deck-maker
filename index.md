## Vault Structure

- `projects/`: 고객사별 deck 작업 폴더 (assets, drafts, html, pptx, docx, pdf)
- `shared-assets/`: 공통 로고, 아이콘, 다이어그램, 배경
- `style-guides/`: 시각 스타일 및 제작 표준
- `deck-templates/`: 덱 유형별 구조 템플릿
- `output-rules/`: HTML/PPTX/DOCX/PDF 출력 규칙
- `scripts/`: Python 유틸리티 스크립트
- `fonts/`: 프로젝트 공통 폰트
- `master-context-usage/`: master-context 참조 가이드
- `outputs/`: 최종 납품본 보관
- `archive/`: 완료된 프로젝트 보관
- `AI-Sessions/raw/`: 수정하지 않는 1차 자료
- `AI-Sessions/conversations/`: 세션 인수인계
- `AI-Sessions/wiki/`: 검증된 업무 지식
